Bundler

Bundler is the most popular tool to manage dependencies. To get started, open a terminal window and run:

$ gem install bundler

Create a Gemfile in your project's root directory. You can generate one with:

$ bundle init

Update the contents of the Gemfile with your project dependencies:

source "https://rubygems.org"

gem "tynn", "1.2.0"

Install all dependencies with:

$ bundle install

Bundler still installs all the gems globally, so you need to load your bundled environment before the gems are required.

require "bundler/setup"
require "tynn"

Check Bundler's homepage for more information.
